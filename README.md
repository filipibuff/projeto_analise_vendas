📊 Dashboard de Análise de Vendas

Projeto de análise de dados desenvolvido utilizando Python, SQL e Streamlit, com foco em transformação de dados, geração de métricas de negócio e visualização de informações estratégicas.

O objetivo do projeto é simular um cenário real de análise de vendas, onde dados brutos são tratados e transformados em indicadores de desempenho (KPIs) e gráficos interativos para tomada de decisão.

---

🚀 Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- SQL
- Streamlit

---

📊 Métricas analisadas

O dashboard apresenta os seguintes indicadores principais:

- Faturamento Total
- Total de Vendas
- Ticket Médio

Esses indicadores ajudam a compreender o desempenho geral das vendas.

---

📈 Visualizações

O projeto inclui gráficos para facilitar a interpretação dos dados:

- Faturamento total
- Distribuição de vendas
- Gráfico de participação por produto (Pizza Chart)

Essas visualizações permitem identificar padrões e oportunidades de negócio.

---

⚙️ Como executar o projeto

1. Clone o repositório

git clone https://github.com/seuusuario/seurepositorio

2. Instale as dependências

pip install -r requirements.txt

3. Execute o dashboard

python -m streamlit run app.py
